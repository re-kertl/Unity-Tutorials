    Class Loot

    Describe an InteractableObject that can be picked up and grant a specific item when interacted with.

    It will also play a small animation (object going in an arc from spawn point to a random point around) when the object is actually "spawned", and the object become interactable only when that anim is finished.

    Finally it will notify the LootUI that a new loot is available in the world so the UI display the name.
