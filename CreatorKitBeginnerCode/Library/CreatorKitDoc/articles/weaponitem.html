    Weapon item

    Weapon item are special equipment item that you can equip on your character weapon slot.

    Like the other equipment item They can have minimum stat requirement that stop them from being equipped until the player reach those stats value. They also can have Equipped Effect that allow to add special effect when the item get equipped (like raising the user stats or elemental protection/boost)

    But they have in addition Stats related to attacking (reach, speed, minimum & maximum damage) and an additional list of effects, Weapon Attack Effect, that allow to customize what happen when it damage another character.

    To create a new Weapon item, right click somewhere in your project window (or use the Asset top menu) and choose Create > Beginner Code > Weapon

    Once the new item renamed, you can use the Inspector to define it. The inspector will look like this :

    • Item Sprite : set the sprite that will be used in the inventory for that item. It will also be used to display the object in the world if no world object prefab is set (see below)
    • Item Name : The name of the item that will appear above it in the world and in the tooltip in the inventory
    • Description : The description of the item that will be displayed in the tooltip in inventory
    • World Object Prefab : the 3D object that will be created to show the object in the game world when it is spawned. If this is null, the item sprite will be used instead.
    • Minimum Stats : the minimum strength, agility & defense required to equip that item. If the player don't have the right value, double clicking on this equipment in the inventory won't do anything.
    • Weapon Stats : the stats of the weapon
      • Speed : how faast the weapon attack, it is the cooldown between 2 attack (lower = faster, 0 mean it can attack again as soon as the animation is finished)
      • Minimum Damage: the minimum amount of damage the weapon can do (before adding any boost or strength). Damage Done is a random number between that & Maximum Damage
      • Maximum Damage: the maximum amount of damage the weapon can do (before adding any boost or strength). Damage Done is a random number between that & Minimum Damage
      • Range: the distance at which the weapon can damage a character. If the target character is past that range when the hit should connect, it won't do any damage.
    • Add New Equipment Effect : this is a dropdown with a list of all the Equipped Effect you wrote. When you write a new effect, it will appear here. Select it to add it to the object. It will appear under with its own inspector and a - button to remove it.
    • Add New Weapon Attack Effect : this is a dropdown with a list of all the Weapon Attack Effect you wrote. When you write a new effect, it will appear here. Select it to add it to the object. It will appear under with its own inspector and a - button to remove it.
