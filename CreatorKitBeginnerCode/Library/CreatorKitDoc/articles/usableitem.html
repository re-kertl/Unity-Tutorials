    Create New Usable Item

    A usable item is an item that can be used by double clicking on it in the inventory.

    We will go through how to create a new type of item here. For how to write the actual effect code that will be execute when the object is use, see Writing a new Usage Effect

    To create a new item, right click somewhere in your project window (or use the Asset top menu) and choose Create > Beginner Code > Usable Item

    This will create a new asset that is the new item.

    The Inspector will look like this :

    • Item Sprite : set the sprite that will be used in the inventory for that item. It will also be used to display the object in the world if no world object prefab is set (see below)

    • Item Name : The name of the item that will appear above it in the world and in the tooltip in the inventory

    • Description : The description of the item that will be displayed in the tooltip in inventory

    • World Object Prefab : the 3D object that will be created to show the object in the game world when it is spawned. If this is null, the item sprite will be used instead.

    • Add New Effect : this is a dropdown with a list of all the Usage Effect you wrote. When you write a new effect, it will appear here. Select it to add it to the object. It will appear under with its own inspector and a - button to remove it.
